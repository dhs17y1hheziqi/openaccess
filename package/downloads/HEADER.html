JDO Genie is a high performance JDO implementation for relational databases. JDO Genie implements all the required features and most of the optional features of version 1.0 of the specification.

For more information visit http://www.hemtech.co.za/jdo

The .zip files are for Windows and the .tar.gz files are for Linux and UNIX. Each archive contains exactly the same files. You will also need JDK 1.4.x or 1.3.x for your platform.

These files will not work without an evaluation license.
